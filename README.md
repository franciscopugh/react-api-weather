# React-api-weather
_First application created with react using a weather api_

### Pre Requirements 📋

```
npm i bootstrap react react-dom react-scripts
```
# Running the tests ⌨️

```
- npm run start
```
_Then go to your browser and put in the search engine:_
```
http://localhost:3000/
```
